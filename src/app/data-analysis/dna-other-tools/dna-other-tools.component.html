Consurf Server Consurf Server uses a Bayesian inference method to calculate the evolutionary conservation of each residue in the protein.The conservation scores are then mapped onto the protein structure to identify the conserved regions. Link

It also supports conversion between different versions of the same identifier, allowing users to track changes in gene and protein annotation over time. Link
Linkage Disequilibrium Calculator A Linkage Disequilibrium (LD) calculator is a tool used in population genetics to estimate the degree of association or correlation between alleles at two or more loci. LD is a measure of non-random association between alleles at different loci, and is influenced by factors such as genetic drift, mutation, migration, and natural selection. Link
